# Sabri Product Data Pipeline

## Overview
This project applies an EVTL pipeline to product data from an external API.

## Data Source
https://dummyjson.com/products

## Pipeline Steps
- Extract data from API
- Validate JSON structure
- Transform into structured dataset
- Load into CSV file

## Output
- Raw: data/raw/sabri_raw.json
- Processed: data/processed/sabri_processed.csv

## Key Insight
Product price does not always correlate with rating.
